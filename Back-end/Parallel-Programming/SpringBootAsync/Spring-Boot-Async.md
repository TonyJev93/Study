- 비동기 vs 논블록킹
    - `비동기`와 `논블록킹`에 대한 비교 기준은 개발자마다 다름
    - `비동기`는 기능을 제공하는 입장에서의 개념이고, `논블록킹` 사용하는 클라이언트 입장에서의 개념
    - 두 개의 입장은 엄연히 다르기 때문에 비교 대상이 될 수 없다.
